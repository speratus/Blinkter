.. currentmodule:: blinkter

Pixel
=====
Represents a single pixel of the Pimoroni Blinkt! board.
In the context of blinkter, a pixel is a set of three LEDs that are associated together
by the blinkt library.

This class contains most of the methods used in operating on individual pixels of the blinkt board.
Most of the common features that users will wish to use are located in this class.
